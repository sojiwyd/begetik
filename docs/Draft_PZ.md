# Черновик пояснительной записки

## Тема проекта
Веб-сервис для автосервиса (запись и учет).

## Актуальность
В 2026 году растет потребность в цифровизации сервисных услуг малого бизнеса, включая автосервисы. Во многих мастерских запись клиентов ведется вручную или через мессенджеры, что приводит к накладкам по времени, потере заявок и отсутствию прозрачной истории обслуживания.

Разработка веб-приложения для автосервиса позволяет автоматизировать процесс записи, контроль загрузки боксов и управление услугами. Проект снижает влияние человеческого фактора, упрощает работу администратора и повышает удобство для клиентов.

## Объект исследования
Процесс организации записи клиентов и учета услуг в автосервисе.

## Предмет исследования
Веб-приложение для автоматизации записи, управления услугами и контроля статусов заказов в автосервисе.

## Цель работы
Разработать веб-приложение для автосервиса, обеспечивающее безопасную авторизацию, управление услугами и записями, а также визуальный контроль занятости слотов.

## Задачи
1. Провести анализ предметной области автосервиса и пользовательских ролей.
2. Спроектировать базу данных для сущностей users, services, orders.
3. Реализовать клиентскую и административную части приложения.
4. Реализовать полный цикл CRUD для услуг и управление заказами.
5. Внедрить защиту от SQL Injection, XSS, CSRF, IDOR.
6. Реализовать календарь занятости и API выдачи слотов.
7. Провести функциональное и безопасностное тестирование.

## Краткое описание реализации
- Авторизация и регистрация пользователей реализованы с хешированием паролей.
- Реализовано разграничение прав доступа (admin/client).
- Для услуг реализованы создание, просмотр, редактирование, удаление.
- Реализованы поиск и пагинация для оптимизации работы со списками.
- Реализована загрузка изображений услуги и фото повреждения автомобиля.
- Реализован личный кабинет пользователя с просмотром только своих записей.
- Реализован календарь занятости и endpoint `api/get_slots.php`, возвращающий JSON со слотами.

## Индивидуальная фишка
Календарь занятости боксов с визуальной индикацией (красный - занято, зеленый - свободно) и API-интерфейсом для получения слотов на выбранную дату.

## Результат
Создан рабочий прототип веб-приложения автосервиса с базовыми бизнес-процессами, реализованной безопасностью и готовностью к дальнейшему масштабированию.
